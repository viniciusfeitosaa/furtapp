# Furtapp — Dr. Francisco Furtado

Site institucional + área do cliente (admin e paciente) para tricologia e transplante capilar.

## Documentação viva

Tudo o que é feito, pendente ou sugerido deve ser registrado em **[MAPA DE BORDO.md](./MAPA%20DE%20BORDO.md)**.

## Stack

- Next.js (App Router) + TypeScript + Tailwind CSS
- Sem WordPress — portal próprio (`/admin`, `/paciente`)

## Desenvolvimento

```bash
npm install
npm run dev
```
