# Manual de identidade — Criativo Submerso (resumo operacional WEB)

Fonte: Manual da Identidade Visual Dr. Francisco Furtado (19.12.2024).

## Diretriz de uso no site (atual)

Objetivo visual: **seriedade, confiança e profissionalismo** — acabamento de site premium.
**Sem degradês.** Apenas planos de cor sólidos e tipografia.

### Planos principais

| Plano | Hex | Uso |
|-------|-----|-----|
| Ink | `#000000` | Hero, header sólido, footer |
| Navy | `#1a2035` | Seção escura secundária (ex.: pilares) |
| Page | `#ffffff` | Conteúdo |
| Quiet | `#ededed` | Alternância leve |
| Gold | `#b6a46e` | CTA / filete de destaque |

### Evitar

Degradês; `#82c4d1` `#a5e1ed` `#c2dadf` em superfícies.
