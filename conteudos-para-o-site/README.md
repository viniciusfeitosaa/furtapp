## Formatos preferidos

- **Fotos:** JPG ou PNG (nós convertemos para WebP). Ideal acima de 1920px no maior lado para hero.
- **Logo:** PNG com fundo transparente **e/ou** SVG.
- **Vídeo:** MP4 (H.264), até ~100–200 MB se possível; se for maior, avise antes.
- **Textos:** `.md` ou `.docx` deixam a edição mais rápida.

## Não coloque aqui

- Fotos **clínicas privadas** de pacientes do portal (M0–M12) — isso é dado médico e vai para storage seguro, não para o site público.
- Senhas, tokens ou dados de cartão.
